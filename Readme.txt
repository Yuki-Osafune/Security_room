・このファイルの中身について

　こちらは、脱出ゲーム、Security room という作品になっております。大学時代の卒業制作の一つとして作成し、脱出ゲームにセキュリティの概念を加えた作品になったいます。
一人でかつ、学業外で自主的に作成し、数か月かけて作成しました。

　初めにゲームの内容をメモにまとめ、どの様に推移するか、何を手に入れて何を調べるとイベントが進むかを記述しました。いわゆる仕様書をあらかじめ作っておく工程を踏んだ形になります。

　実際に作る際、部屋の推移やアイテム欄などの実装をする際、分からない部分を積極的に調べて制作していました。その過程で、調べた内容を応用し自動でイベントを進むシーンを作成できた辺りで手ごたえを感じ始め、このあたりから制作速度の上昇を実感しました。

反省点としては、自身の実力が高くなく、実装できずに妥協せざるを得ない点があった事や、一部軽度の不具合が直せなかった事や、時間の都合でエンディングが簡素なものになってしまったなど、いくつか挙げられます。
